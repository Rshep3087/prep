# prep

TUI for browsing and running mise tasks.

## Install

```
go install github.com/rshep3087/prep@latest
```

Or build from source:

```
mise run build
```

## Usage

```
./prep
```

Navigate with arrow keys, Tab between sections (Tasks, Tools, Env), Enter to run a task, q to quit.

## Requirements

- mise
- Go 1.25+
